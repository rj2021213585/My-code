## 最后一个单词的长度
```c
int lengthOfLastWord(char * s)
{
    int right  = strlen(s)-1;
    int size = 0;
    for(;right>=0;right--)
    {
        if(s[right]!=' ')
        {
            size++;
        }
        if((s[right]>='a'&&s[right]<='z'||s[right]>='A'&&s[right]<='Z')&&right-1>=0&&s[right-1]==' ')
            break;
    }
    return size;
}
```
